🔧 해결 방법

브라우저에서 페이지가 열리지 않는 경우:

  1. 캐시 클리어: Ctrl+F5 또는 Ctrl+Shift+R
  2. 개발자 도구: F12 → Console 탭에서 오류 확인
  3. 로컬 서버 사용: file:// 대신 http://localhost 사용
  4. 다른 브라우저: Chrome, Firefox, Edge 등으로 테스트
